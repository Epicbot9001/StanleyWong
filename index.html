StockWise

Neural Network and LSTM model for predicting NASDAQ stock movements.

I developed machine learning models to predict NASDAQ stock price movements using Keras and TensorFlow, leveraging the frameworks for efficient building, training, and testing of neural networks. I implemented a Sequential Neural Network model with multiple Dense Layers, ReLU activation, and L2 regularization, achieving a 52% accuracy in predicting stock movement direction.

Additionally, I designed and tested an LSTM model, converting 2D data to 3D format to better capture time-series dependencies, which resulted in improved performance with a Mean Absolute Error (MAE) of 5.376 compared to 5.433 for the simple neural network.

To evaluate and guide model improvements, I employed metrics such as Mean Squared Error (MSE), Mean Absolute Error (MAE), and Absolute Percent Difference. Throughout the project, I overcame challenges like the unpredictable nature of stock prices and the risk of overfitting by refining the model architecture and testing various approaches.

View this project here: https://stockwise-99.streamlit.app/

Ceptron

A virtual assistant with a suite of machine learning models.

Ceptron is a project created for HackGT 9, prior to the release of ChatGPT. I led a team to create Ceptron to make it easier for users to leverage machine learning models directly from their workflow with a simple keyboard shortcut, Ctrl+Shift+C. Ceptron enables quick invocation of ML models, copying their outputs directly to the clipboard for easy pasting into any document.

I built this project to support three main functions: summarizing text from a URL, generating images using Stable Diffusion, and providing natural language responses with GPT-3. I implemented the backend on Google Cloud Platform (GCP), using Cloud Run for less resource-intensive tasks like text summarization (with the sumy model), Compute Engine for image generation with Stable Diffusion, and Firebase for routing and static hosting. The frontend, developed with GitHub Electron, captures global shortcuts and manages the command execution. Despite challenges like deploying large ML models on a virtual machine, I successfully created a cross-platform desktop application, providing users with a simple yet polished interface for interacting with advanced machine learning models.

Deepfake Detection

Deepfake detection model using hybrid CNN-RNN architecture for robust video analysis.

The image above shows the diagram of our deepfake detection model. The images pictured on the top left represent the different modifications we added to the test set to test for robustness.

I built a robust deepfake detection model using a hybrid CNN-RNN architecture, focusing on identifying manipulated videos even under various real-world distortions such as noise, resolution changes, facial obstructions, contrast variations, and shadowing. Leveraging tools like Python, NumPy, Scikit-learn, and TensorFlow, I executed experiments on 400 videos from the Deepfake Detection Challenge (DFDC) dataset, achieving up to a 5% accuracy improvement over baseline models.

My work not only evaluated the model's performance under these conditions but also contributed valuable insights into the robustness and adaptability of deepfake detection systems. This project showed the importance of enhancing detection capabilities to maintain the authenticity of digital content amidst the growing threat of advanced synthetic media.

ReVision

AR memory game mobile app with image recognition and real-world integration.

ReVision is an app created for HackGT X, Georgia Tech’s largest annual hackathon with participants from around the nation. I collaborated with a team of four to produce ReVision, an augmented reality (AR) app that allows users to visualize images and engage in a memory game by integrating digital content with real-world environments.

Photos are selected by the user to be in the game, and location metadata is extracted from every photo. When the user starts the round, they will be shown four photos and one location, and they have one minute to choose as many matching picture-location pairs as possible. ReVision also allows participants to view their photos in an AR space, one tap is all that is required for a picture to show up in AR.

Using ARKit, Swift, and SwiftUI on XCode, I focused on creating a seamless and immersive user experience. I enhanced AR rendering to ensure consistent performance across various devices and designed and implemented image recognition and matching algorithms, which elevated user engagement through the app's interactive features.
